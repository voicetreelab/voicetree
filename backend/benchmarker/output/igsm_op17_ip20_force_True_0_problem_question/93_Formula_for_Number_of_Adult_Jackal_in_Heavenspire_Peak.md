### Calculates the number of adult maned jackals in Heavenspire Peak by summing adult parrots and blue jays in South Zoo.

The number of adult maned jackal in Heavenspire Peak equals the sum of the number of adult parrot in South Zoo and the number of adult blue jay in South Zoo.

_Defines:_
- number of adult maned jackal in Heavenspire Peak

_Requires:_
- number of adult parrot in South Zoo
- number of adult blue jay in South Zoo

### Calculates the average number of newborn children per adult booby in Lunarchasm Ridge based on crow and eagle populations.

The average number of newborn children per adult booby in Lunarchasm Ridge equals the difference between the average number of newborn children per adult crow in South Zoo and the average number of newborn children per adult eagle in Mayer Aquarium.

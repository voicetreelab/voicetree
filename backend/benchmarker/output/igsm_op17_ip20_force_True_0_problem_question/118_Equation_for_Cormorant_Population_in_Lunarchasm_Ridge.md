### Calculates the number of adult cormorant in Lunarchasm Ridge based on adult parrot and adult eagle populations.

The number of adult cormorant in Lunarchasm Ridge equals the sum of the number of adult parrot in Mayer Aquarium and the average number of newborn children per adult eagle in South Zoo.

_Defines:_
- number of adult cormorant in Lunarchasm Ridge

_Requires:_
- number of adult parrot in Mayer Aquarium
- average number of newborn children per adult eagle in South Zoo

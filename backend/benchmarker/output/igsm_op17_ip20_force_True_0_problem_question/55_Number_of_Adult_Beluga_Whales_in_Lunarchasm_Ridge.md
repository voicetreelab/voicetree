### Defines the number of adult beluga whales in Lunarchasm Ridge based on newborn populations of crows and blue jays in South Zoo.

The number of adult beluga whale in Lunarchasm Ridge equals the sum of the average number of newborn children per adult crow in South Zoo and the average number of newborn children per adult blue jay in South Zoo.

_Defines:_
- number of adult beluga whale in Lunarchasm Ridge

_Requires:_
- average number of newborn children per adult crow in South Zoo
- average number of newborn children per adult blue jay in South Zoo

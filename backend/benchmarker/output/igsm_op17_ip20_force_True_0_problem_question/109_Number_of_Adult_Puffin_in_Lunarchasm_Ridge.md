### Defines the total number of adult puffins in Lunarchasm Ridge based on eagle and blue jay populations.

The number of adult puffin in Lunarchasm Ridge equals the sum of the number of adult eagle in Mayer Aquarium and the average number of newborn children per adult blue jay in Mayer Aquarium.

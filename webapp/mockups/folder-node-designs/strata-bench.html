Folded assembly shows 3-port interface and item count. Spec sheet still openable from collapsed state.

nesting language

ANSI Y14.2 line conventions: L1 = continuous double rule, L2 = dashed phantom outline, L3 = dotted detail outline. Depth reads from line weight + dash pattern, never paper tint alone.

item numbering

Each child gets a circled item number (1, 2, 3...) that survives into the title-block BOM. Color of the roundel encodes status: white = done, blueprint = spec, vermillion = attention.

spec sheet + inspect

Spec-sheet icon is the same sheet-with-fold glyph. The toggle is rendered as [−]/[+] bracket pairs — engineer's shorthand for inspect/fold. Toggle inverts to ink-on-paper when active.

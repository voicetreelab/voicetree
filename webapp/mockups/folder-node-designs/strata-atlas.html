Collapsed sheet exposes its edge interface as a 3-port fan on the right border. Cross-folder edges meet the fan, never the absent members.

nesting language

Three hatching directions per level — +45° / -45° / 90° — make depth legible without color. Inner folders inherit the parent ground but add a darker paper tint (paper-2, paper-3) so depth = paper weight.

control bar

Each folder publishes one 30px-tall strip with the same five slots: L# chip · name · bearing/count · note · toggle. Slot positions never change; only their content shrinks at deeper levels. Toggle becomes inverse (ink-on-paper) when collapsed.

folder-note icon

The note glyph is a sheet-with-corner-fold — the same iconography as architectural callout sheets. Clicking opens foldername/foldername.md per the BF-108 identity contract. Glyph color is blueprint cyan so it reads as interface, not status.

Sealed package exposes only its pin rail on the right edge. Traces from other dies terminate at the rail, not at internal nets.

nesting language

Each level is a chip in a chip in a chip. Designators stack hierarchically — U1 → U1A → U1A1 — so depth is encoded in the name itself. DIP pin rails on top/bottom of every package; cyan rails mark the L1 outer IC.

copper interface

External edges become right-angle copper traces with circular pads at every junction. Trace width is consistent (no per-edge weight); pads handle the visual hierarchy. Internal nets are faint dotted, never solid.

datasheet + switch

Folder-note glyph is labeled datasheet here — same sheet-with-fold icon, electronic context. Toggle is a TTL-style switch: outlined oval, dot drifts left (off / collapsed) or right (on / expanded). A real PCB designer would feel at home.
